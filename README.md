# Doppio — releases & update feed

This public repo hosts the **[Sparkle](https://sparkle-project.org) update feed**
(`appcast.xml`) and notarized **DMGs** for [Doppio](https://github.com/ihavespoons/doppio),
a macOS menu-bar app that keeps your Mac awake — even with the lid closed.

- App source lives in the private `ihavespoons/doppio` repo.
- Releases here are published automatically by CI on each version tag.
- The app's `SUFeedURL` points at `appcast.xml` on this repo's `main` branch.
